Conceito DIV

O conceito do DIV consiste em dividir o nosso site em blocos como por exemplo: Topo, Menu, Conteúdo 1 e Conteúdo 2.
A tag de DIV é <-div><-/div>.obs:Sem os -. Podemos utilizar ela da maneira que está demonstrada aqui abaixo:
A div sozinha não serve para muita coisa utilizamos ela para que fique mais fácil a organização do projeto HTML integrado com o CSS, por isso utilizam as div's juntamento com o HTML semântico, que consiste em trocar algumas div's por tags que tem o mesmo efeito com outro nome e deixa o nosso código mais fácil de ser interpretado. Como por exemplo:
Mudar o nome da div do topo para header que significa topo, assim funcionando do mesmo jeito mas na hr da leitura mais fácil de ser interpretado.
Outro exemplo: no menu, podemos trocar a tag div para nav.
Mais um exemplo: podemos trocar as div's do conteudo pelo nome section.
Tambem podemos utilizar a tag footer ao inves de div quando formos fazer o rodapé de nosso site.
