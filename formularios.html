Formulários

Algumas das tags mais usadas para fazer Formulários são:

  • <-form><-/form> - Tag Pai, Nosso formulário sempre deve estar dentro dessa tag.
  • <-label for="nome_id"> "Aparece no Site" <-/label>.
  • <-input id="Voce_escolhe" type="text"(caixinha de texto) placeholder="legenda_caixinha">.
  • <-input id="Voce_escolhe" type="password"(Caixinha de texto, porem oculta o conteudo) placeholder="digite sua senha">.
  • <-input id="Voce_escolhe" type="email"(caixinha de texto, porem reconhece quando email. Precisa por @ e .) placeholder="digite seu email">.
  • <-input type="radio"(Bolinha de seleção, escolhe somente 1) id="Voce_escolhe" name="Voce_escolhe" value="Voce_escolhe"(será enviado pelo formulário para o BD)>.
  • <-input type="checkbox"(caixinha de seleção, pode selecionar vários) id="Voce_escolhe" name="Voce_escolhe" Value="Voce_escolhe"(será enviado pelo formulário para o BD)>.
  • <-input type="submit"(caixinha de enviar formulario) value="Voce_escolhe"(será enviado pelo formulário para o BD)>.
  • <-select>(Open pai) <-option value="Voce_escolhe"(Será enviado pelo formulário para o BD)>"mensagem_que_será_mostrada"<-/option> <-/select>(close pai).
  • <-option selected(seleciona como principal) disabled(disabilita para seleção) value="deixa vazio">"o_que_a_Pessoa_Tem_que_escolher"<-/option>.
  • <-textarea required(campo precisa ser preenchido) name="Voce_escolhe" rows="10"(linhas) cols="60"(colunas) placeholder="legenda_caixinha"><-/textarea>
  • <-button>(adiciona um botão qualquer)"nome_do_botao"<-/button>
obs:Sem os -.
